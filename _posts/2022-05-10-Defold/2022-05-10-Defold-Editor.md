The defold editor has everything you need but is not very intuitive. So to help you a little *F6, F7, F8* can be used to toggle panes to give more space. When searching with Ctrl-F etc. *note* click the search box and press *ESC* to close it. 

There are several panes or panels in the editor.

+ Assets - you can create files or drag in files. 
+ Outline - you can edit collections and Game Objects.
+ Properties - Values that affect the current object 
+ Tools - mostly we'll focus on the console output
+ Editor - This is where we view our scene and view our code.

![](/img/editor-images/firstLookAtMainCollection.png)

### **Important keys**
+ use CTRL+Space for auto completion in the code view.
+ F6, F7, and F8 show/hide parts of the view to avoid crowding.
+ F5 to Run/Debug the app during development

The Editor pane

    The center view shows the currently open file in an editor for that file type. All visual editors allows you to change the camera view:
