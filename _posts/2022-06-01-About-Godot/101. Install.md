## Install Godot

Requirements are listed in the introduction in this book or on the following platforms.

There are a few ways to get *Godot* and its editor. The two most common and very easy ways are listed here. (links at the bottom of the page)

### Steam:

Yes if you already use the Steam app platform then you can simply search for Godot and install it from here. Its free so there is no need to pay for it.

### Godot:

The defold website https://godotengine.org/download allows you to freely download its software and contains example, tutorials, documentation and a great community that you can ask questions. 
Download the latest standard version. Once downloaded run the installer.
We don't cover the mono version in this book. Gdscript is more beginner friendly than C#.

## Run first time

Run the newly installed app. It should take less than a minute. 

Click on New Project on the right side, rename your project. (note where the project will be saved). You may want to browse to another location, but default is ok. Click on Create Folder. All the other defaults are ok, unless you plan on only doing web apps, then change to OpenGL ES 2.0.

![](/img/install/FristRun.png)

Your screen may not have your own or example projects yet, but soon you will have. If you got the app from Steam the examples are included.

Click Create & Edit
 


![](/img/install/GodotEmptyProject.png)


Select 2D from the Top menu and then Create a 2D scene on the left. Use Ctrl S to save and save as scene1.tscn. 


![](/img/install/create2dSceneAndRun.png)

Click the circled symbol to run.

![](/img/install/selectCurrentScene.png)

select current

![](/img/install/emptyGame.png)
 
Note the name on the top left - if that shows something else on your screen its likely you forgot to name your project, or named it something else.
To close, click on the x, or just close the editor.
When running defold again if you closed the editor you will see the newly created project in the Local Project, but may have to search for it if you have examples. Click the star on the left of the name to place it at the TOP.

![](/img/install/placeAtTop.png)
