## Godot - Editor:

Let's take a Quick look at the Godot editor and the various panes.

Open Godot, open the HeroRPG project in "Local Projects". Select Project and Project settings from the menu.

![](/img/Editor/HeroRPG_projectSettings.png)

There is a long list of settings. I want you to be aware of this as you will need to make changes here at some point. Usually on the display size. 

Notice properties has the main scene set as scene1 i.e. the one we just created.

For now we will leave it alone as we are happy with the default for the Desktop project.

the docs for the editor can be found at 


![](/img/Editor/EditorPanels.png)

+ 2 = Viewport with its toolbar at top. The line in the diagram allows us to switch between 2D, 3D, script, and asset-lib. The asset library gives us add-ons. Clicking those also changes the toolbar which is specific to the viewport.
+ 5 = Files
