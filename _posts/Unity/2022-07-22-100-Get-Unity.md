## Unity Install

When you begin, it is often best not to make tto many changes untill you become comfortable with the game engine.
So ...

Let's go to the Unity web-site and choose the latest LTS (long-term-service) version. I believe it should be valid 3 years from its initial release date.
The link for student or individual plans (freeish) is https://store.unity.com/#plans-individual or https://unity3d.com/get-unity/download. If this changes search for "Unity 3d game engine" and then look for the free version.

![](get-started-images\download-individual.png)

click get started and then find the HUB for your operationg system i.e. download for windows.

The Hub is a piece of software which allows you to download several game engines. We just want one for now.
Go ahead Download the Hub.

![](get-started-images\for-windows.png)

Find the hub in the downloads folder and install it.

Run it and go to the installs tab.

![](get-started-images\hub.png)

Click install editor

![](get-started-images\hub-install-editor.png)

Select your version and then select the modules you want to build for. I suggest you just stick to the one you are using for now, but if you plan on android (or others) you can get that module too.

Make sure that the dev-tools Visual Studio (or VS-Code) is selected. I wont go into alternates here, best check whith their documentation and tips for getting it to work. We just use defaults here.

It takes a while to install. When its done you will want to open VS first - for some reason it needs to run at least once. It sometimes comes with a HUB too and from there you can select the support you need (if different from what Unity chose for you - anyway you should get C# and intellicode automatically.)

While this is running lets get ourselves set up for our first project by getting GIT.
