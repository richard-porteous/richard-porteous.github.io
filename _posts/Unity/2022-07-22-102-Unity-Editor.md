## A Look at the Editor

Assuming your Unity and Visual Studio have downloaded and installed we can begin.

Visual studio is used simply as a code editor - so something lighter can be chosen if you could find alternative install instructions.
If give intelligent help with syntax and code suggestions. If tht is not working when you try it you may nedd to do some trouble shooting.

Lets Start.

Open Unity Hub. You will see starting here avoids some problems later.

![](get-started-images\create_a_project_p1.png)

Click new project

![](get-started-images\create_a_project_p2.png)

We will choose 2D core and then normally we would change the name *new project* at the bottom right to something more useful. But... we can leave that for now.

if you have created a specific folder you will want to make sure that the folder you want for the project is created is selected. Otherwise click *create project*.
