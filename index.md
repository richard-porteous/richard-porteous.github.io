Hi, I’m @richard-porteous

👀 I’m interested in most things technical. Solar panels, robotics etc. Transferring knowledge to new things, and learning new ways to do the same thing, hopefully better.

🌱 Trained in electronics I’m currently learning for a Computer Science Degree. Already a software developer. #living-life-backwards. most of my life has been self directed learning.

💞️ I’m building a knowledge portfolio, something which I have neglected in the past.


I'm going to start with coding, essential concepts behind certain game engines and other things. I'm hoping not to take the learning experience away but help break some of the blockages that so many people get snagged on (including myself). I may also take a look at electronics kits too … but it is still early.

The site in its current state is **still in beta** and I'm mostly focusing on presentation as I want this to be a place where anyone can learn and not simply give up because its too wordy.

This site is built in GitHub pages. I used a template from [ChadBaldwin](https://chadbaldwin.net/2021/03/14/how-to-build-a-sql-blog.html "ChadBaldwin"). I use [Atom](https://atom.io/ "Atom") as my markdown editor. I use the built in git in Atom or a Git Client UI [GitAhead](https://gitahead.github.io/gitahead.com/ "GitAhead") to commit all the changes I made, then push back to GitHub.
