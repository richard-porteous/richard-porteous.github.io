Using microsoft powershell

Date: 20.01.2024 - Author: Richard Porteous

Microsoft Powershell is NOT Windows Powershell

I’ve never liked windows powershell while others have sworn by it. I stuck with cmd and ignored windows powershell with a vengance. OK, OK, I did try it for a month or two, but found it slowing me down and in places being an obstruction. Perhaps it was more “when” and “what”, but I wasn’t happy.

So why would I try Microsoft Powershell? Well, I discovered they are different. A little late you say? Yes well thats what happens when you close your mind.

I discovered it when first trying Hugo a static site generator and they explained that on windows that cmd and windows powershell were not recommended and that I should use WSL, a linux terminal, or Microsoft Powershell 7+

My quick take

Ive not used it enough yet but Its a power cmd. It doesn’t get in the way or force you to do things differently. It had beneficial features like pulling up a history list with F2 and completing a line with F8 based on your history of commands. So you can use it like cmd but still benefit from the exta features at your own pace.

Pretty much I like it enough to write a post about it.
